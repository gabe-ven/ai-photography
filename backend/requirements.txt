# Subject localization (open-vocab detection + segmentation refinement).
# Heavy: ultralytics pulls in torch and downloads YOLO-World weights on first
# use. The composition pipeline degrades gracefully to the saliency centroid
# when these are unavailable, so tests/CI run without weights or a GPU.
#
# Version notes:
# - ultralytics >=8.4 drops the legacy `numpy<2` cap, so it coexists with the
#   numpy 2.x pin above (8.3.x forces numpy<2 and conflicts).
# - torch/torchvision are pinned as a matched pair; CLIP depends on an
#   unpinned torchvision and will otherwise pull a build that expects a newer
#   torch than we install.
# - YOLO-World's open-vocab head scores boxes against text prompts via CLIP,
#   so `model.set_classes(...)` requires the CLIP package below. Without it,
#   ultralytics raises `ModuleNotFoundError: No module named 'clip'` and the
#   pipeline falls back to the saliency centroid.
